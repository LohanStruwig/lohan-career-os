## Privacy And Safety

This is a public repository. It intentionally excludes private contact information and sensitive personal details.

Do not add:

- phone numbers
- street addresses
- personal identity documents
- credentials, tokens, API keys, or environment files
- private client data
- confidential employer material
- non-public financial, customer, vendor, or employee information
- private screenshots or internal documents

Public-facing career facts should be written at a professional level of abstraction, backed by evidence, and safe to share.

## How To Use This Repository

Use `master-cv/` as the source of truth, then pull from `resume-blocks/`, `capability-blocks/`, and `evidence-ledger/` when tailoring a resume or building a portfolio page.

Recommended workflow:

1. Update the relevant experience or capability file first.
2. Add supporting proof to `evidence-ledger/`.
3. Convert the strongest version into a shorter reusable block in `resume-blocks/`.
4. Reuse the block in resumes, LinkedIn, websites, case studies, or interview prep.
